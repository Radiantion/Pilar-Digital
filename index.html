Criando um Mundo na Internet

A eletiva de Desenvolvimento Web tem como objetivo ensinar os alunos a criarem sites simples, entendendo como funcionam as páginas da internet. Durante as aulas, aprendemos os conceitos básicos de construção de sites usando HTML e CSS, desenvolvendo projetos práticos para aplicar o que foi aprendido.

Apenas o básico

O que é HTML

O HTML é a base de todo site. Ele serve para organizar e estruturar os conteúdos, como textos, imagens, links e listas. Com ele, conseguimos montar o esqueleto da página e definir o que vai aparecer na tela.

O que é CSS

O CSS é usado para deixar o site bonito e com estilo. Ele permite mudar as cores, fontes, tamanhos, posicionamentos e criar layouts mais agradáveis. É como se fosse a "roupa" do site, deixando tudo mais visual e organizado.
